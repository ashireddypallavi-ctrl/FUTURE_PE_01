# FUTURE_PE_01
# AI Website Copy Generator for Local Businesses

## 📌 Project Overview

This project demonstrates how **prompt engineering** can be used to generate high-quality, website-ready content for local businesses using AI.

The task focuses on creating structured prompts to generate **professional and trustworthy website copy** for a cafe/restaurant business, covering:

* Homepage
* Services/Menu Page
* Call-to-Action (CTA) Sections

The outputs are designed to be **conversion-focused, clear, and ready for real-world use**.

---

## 🧠 Prompt Engineering Approach

The prompts were designed using a structured framework:

* **Role Assignment**
  AI is instructed to act as a professional website copywriter.

* **Business Context**
  Includes business type (cafe/restaurant), location (Hyderabad), and target audience.

* **Tone Specification**
  Professional and trustworthy tone maintained across all outputs.

* **Output Structure**
  Clear instructions for headings, paragraphs, and formatting.

* **Constraints**
  Word limits and clarity guidelines to ensure concise, usable content.
